html

  • Creates the structure of webpage. 1 of 3 Key Languages used in web development.
  • Made up of elements that make the framework: Elements to add content is text elements, image elements, header elements, and list elements.
  • Elements can also be used to divide content and describe content to browser and other developers. Some of those elements anre Meta-related elements, Content divison elements, and Semantic elements.
  • Must build a sturdy "Frame". Build from ground up.
  • Use Series of HTML elements to define content and framework.
  • Elements are created in opening and closing tags by using the angle brackets.
  • 2 Elements that are important to how structure works and the "head" and the "body"
  • the "head" contains info or metadata that helps the browaser to render the page correctly.Elements in this section are not visible when users look at page.
  • the "body" contains content shown to users. In this section users can see and interact with page.

CSS

  • CSS stands for Cascading Styling Sheet
  • it gives the content style.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

JavaScript
